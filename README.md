# 🚀 Global Space Missions Analysis (1957–2022)

An end-to-end Data Analytics project that analyzes over 4,500 global space missions conducted between 1957 and August 2022 using **Python, MySQL, Power BI, and DAX**.

---

# Project Overview

This project follows the complete data analytics lifecycle:

Raw Data
→ Data Cleaning
→ Feature Engineering
→ Exploratory Data Analysis
→ SQL Business Analysis
→ Interactive Power BI Dashboard

The objective is to uncover historical trends, evaluate mission performance, and generate actionable business insights through data visualization and reporting.

---

# Dataset

- **Time Period:** 1957 – August 2022
- **Total Missions:** 4,500+
- **Companies:** 61
- **Rockets:** 358
- **Countries:** 21

# Key Business Insights

- Mission success rate exceeds **90%**.
- Launch frequency has increased rapidly since 2015.
- The United States and Russia dominate global launch activity.
- Commercial companies are driving modern space exploration.
- A small number of organizations perform the majority of launches.
- Launch costs vary significantly across companies and rocket families.
